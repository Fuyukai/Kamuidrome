Welcome to Kamuidrome's documentation!
======================================

*Kamuidrome* is a command-line application to assist in the development of Minecraft modpacks.

Installation
------------

Kamuidrome is available on either PyPI or via GitHub and is installable with `pipx`_.

.. code-block:: fish

    $ pipx install kamuidrome
    # or alternatively
    $ pipx install git+https://github.com/Fuyukai/Kamuidrome.git
